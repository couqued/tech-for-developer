### Calendar 클래스를 이용한 날짜와 시간 구하기
- java.util 패키지 사용
- Java 1.1 (1997년 등장)
- MONTH + 1 을 해야하는 불편함

```Java
public static void main(String[] args) {
    
    Calendar cal = Calendar.getInstance();  //추상클래스로 인스턴스를 생성할 수 없음
    System.out.println(cal.get(Calendar.YEAR));
    System.out.println(cal.get(Calendar.MONTH)+1);
    System.out.println(cal.get(Calendar.DATE));

    System.out.println(cal.get(Calendar.HOUR));
    System.out.println(cal.get(Calendar.MINUTE));
    System.out.println(cal.get(Calendar.SECOND));

    cal.add(Calendar.HOUR, 5);
    System.out.println(cal.get(Calendar.HOUR_OF_DAY));

    // 포맷에 맞는 현재 날짜 시간 출력
    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
    String formatted = sdf.format(cal.getTime());

    System.out.println("현재 시간: " + formatted);

}
```

### LocalDate 클래스를 이용한 날짜, 시간 구하기
- java.time 패키지 사용
- Java8 (2014년 등장)
- 메서드 체이닝 가능

```Java
public static void main(String[] args) {
    LocalDateTime timePoint = LocalDateTime.now();  // 현재 날짜, 시간
    System.out.println(timePoint);                  // 2025-11-09T11:08:55.880069

    LocalDate ld = LocalDate.of(2012, 12, 1);       // 날짜 설정
    System.out.println(ld);                         // 2012-12-01

    LocalTime lt = LocalTime.of(17, 18);            // 시간 설정
    System.out.println(lt);                         // 17:18

    LocalDateTime dt = LocalDateTime.of(2025, 12, 25, 14, 30, 45);  // 날짜, 시간 설정
    DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
    System.out.println(dt.format(formatter));       // 2025-12-25 14:30:45

    LocalDateTime nextDay = dt.plusDays(1);         // 다음날
    System.out.println(nextDay.format(formatter));  // 2025-12-26 14:30:45

    LocalDate theDate = timePoint.toLocalDate();
    System.out.println(theDate);                    // 2025-11-09
    System.out.println(timePoint.getMonth());       // NOVEMBER
    System.out.println(timePoint.getMonthValue());  // 11
    System.out.println(timePoint.getHour());        // 10
    System.out.println(timePoint.getMinute());      // 8
    System.out.println(timePoint.getSecond());      // 55
}
```

- 기타 메서드
메서드	설명
  - plusDays(long days)	    : 지정한 일 수를 더한 새로운 날짜 반환
  - minusDays(long days)    : 지정한 일 수를 뺀 날짜 반환
  - plusHours(long hours)   : 시간 더하기
  - plusWeeks(long weeks)   : 주 단위 더하기
  - plusMonths(long months) : 월 단위 더하기
  - plusYears(long years)   : 연 단위 더하기
